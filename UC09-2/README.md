# SENAI-ATIVIDADES

Esta é a execução da Atividade 2 - UC09,  para o projeto de um site de uma loja de eletrônicos voltado ao universo gamer, e inicio do desenvolvimento prático de seu Front-End. 
Utilizando HTML e CSS3 para marcação e estilização;
Media query e bootstrap para Implementação de responsividade;
JQuery e JavaScript para implementação de funcionalidades.
