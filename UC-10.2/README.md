# LojaGames

Este projeto foi gerado com  [Angular CLI](https://github.com/angular/angular-cli) version 15.0.4.

## Development server

Rode  `ng serve` para o modo dev server. Navegue até a página  `http://localhost:4200/`. A aplicação irá reiniciar automaticamente se você mudar qualquer coisa no arquivos de origem.

## Code scaffolding

Run `ng generate component component-name` para gerar um novo componente. Você pode usar algo como  `ng generate directive|pipe|service|class|guard|interface|enum|module`.

## Build

Rode `ng build` para construir o projeto. O artefato de construçção será gravando no diretório `dist/` 

## Running unit tests

Rode `ng test` para executar um teste único via [Karma](https://karma-runner.github.io).

## Running end-to-end tests

Rode `ng e2e` para executar teste end-to-end com a platforma da sua escolha. Para usar esse comando, você precisa primeiramente adicionar o pacote que implementa a capacidade de teste end-to-end.

## Further help

To get more help on the Angular CLI use `ng help` or go check out the [Angular CLI Overview and Command Reference](https://angular.io/cli) page.
